Создайте калькулятор для введённых значений

https://learn.javascript.ru/task/calculator-for-input Напишите код, который: Запрашивает по очереди значения при помощи prompt и сохраняет их в массиве. Заканчивает ввод, как только посетитель введёт пустую строку, не число или нажмёт «Отмена». При этом ноль 0 не должен заканчивать ввод, это разрешённое число. Выводит сумму всех значений массива
